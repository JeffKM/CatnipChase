## 핵심 정보

**목적**: 강아지 악당 콤비에게 빼앗긴 마법 캣닢 씨앗을 되찾는 8비트 레트로 아케이드 액션 게임
**사용자**: 레트로 게임을 즐기는 캐주얼 게이머, 해당 캐릭터들을 아는 지인

## 씬 플로우 (사용자 여정)

```
1. Title 씬
   - 게임 로고 + "PRESS START" 표시
   ↓ Spacebar 또는 Enter 입력

2. Game 씬 (Stage 1)
   - 횡스크롤 본게임 진행
   - 장애물 3종 회피/파괴, 아이템 수집
   ↓ 스테이지 끝 도달 (선아 미니보스 구간 진입)

   [하트 0개] → GameOver 씬
   [보스 찧기 3회 히트] → Clear 씬

3-A. Clear 씬
   - "STAGE CLEAR" + 최종 점수 표시
   ↓ 아무 키 입력 (또는 일정 시간 후 자동)
   → Title 씬으로 복귀

3-B. GameOver 씬
   - 현재 점수 표시
   - "TRY AGAIN" 선택 → Title 씬
   - "QUIT" 선택 → 브라우저 탭 닫기 (또는 Title 씬)
```

#### 선아 미니보스 패턴 상세

```
[보스 등장]
- 스테이지 끝 지점에서 선아(유모차) 자동 등장
- 유모차를 밀며 좌우 순찰 이동

[패턴 1: 순찰]
- 일정 범위 내 좌우 이동 반복

[패턴 2: 돌진]
- 선아가 플레이어 방향으로 고속 돌진
- 플레이어: 점프로 회피 → 착지 후 찧기로 히트

[피격 처리]
- 찧기(Heavy Thump) 충격파 충돌 시 히트 카운트 +1
- 3히트 = 처치 → Clear 씬 전환

[피격 미성공]
- 돌진에 피격 시 플레이어 하트 -1
```

#### 장애물 배치 원칙 (Stage 1 디자인 가이드)

```
구간 A (도입부):   땅굴 함정 2~3개 → 이동/점프 익히기
구간 B (중반):     쓰레기통 등장 → 점프 회피 연습
구간 C (후반):     불타는 바리케이드 1개 → 찧기 스킬 강제 학습
구간 D (보스 직전): 혼합 배치 → 최종 난이도 체크
구간 E (보스):     선아 미니보스전
```

## 기술 스택

### 게임 엔진 & 프레임워크

- **Phaser 3.87+** — 게임 엔진 (물리, 씬 관리, 렌더러)
- **Next.js 15** (App Router) — 웹 프레임워크 (게임 호스팅 페이지)
- **TypeScript 5.6+** — 타입 안전성 전체 적용

### 렌더링

- **Phaser WebGL 렌더러** — 게임 씬 렌더링
- **해상도**: 256x240 내부 → 브라우저 창에 정수배 스케일 (최대 16:9 레터박스)

### 스타일링 (Next.js 래퍼 페이지)

- **TailwindCSS v4** — 게임 래퍼 페이지 레이아웃 (전체 화면, 중앙 정렬)

### 오디오

- **jsfxr** — 효과음 생성 및 재생 (피격, 아이템, 파괴, 클리어 등 8비트 효과음)
- BGM 없음 (MVP 범위 외)

### 에셋

- **플레이스홀더 도형** — MVP 단계. Phaser Graphics API로 사각형/원으로 캐릭터/장애물 대체
- 픽셀 아트 스프라이트 교체는 MVP 이후

### 배포

- **Vercel** — Next.js 15 최적화 정적/SSR 배포
- `next export` 또는 Vercel 자동 배포

### 패키지 관리

- **npm** — 의존성 관리
